About

Cascade is a minimalist personal vCard template based on four vertical menu tabs. The first tab is a typical descriptive about page with slider, the second is a gallery with build-in lightbox, next tab is a personal info page and the last tab is the contact page with working form. Included PSD source file allows you to customize this template to your needs.

Template features

  • jQuery Powered
  • Build-in Lightbox
  • Working Contact Form
  • Support for back/next browser buttons
  • Latest Tweets
  • Font Face Fonts
  • Valid XHTML Code
  • PSD Source Included
  • Documentation Included

CSS styles

The project uses the following cascade of template styles (in order):

Filename Description
nivo-slider.css default styles for slider
jquery.qtip.css default styles for tooltips (overwritten in base.css)
jquery.captify.css default styles for image labels (overwritten in base.css)
jquery.jScrollPane.css default styles for scrollbars (overwritten in base.css)
fancybox/jquery.fancybox.css default styles for fancybox
base.css base styles for the entire project
page.css styles for subpages

All CSS files are located in /style directory.

HTML structure

HTML page of this project is divided by three section: header, content (main section) and footer.
When the box is opened, script (cascade.js) loads the page specified by the hash of clicked box.
For example, if you clicked About box, the page /page/page-1.html is loaded. All subpages are stored in /page directory.

After the page is loaded the script page/script/main.js is running. Additionally for each page can be loaded locally JS script from path page/script.

Configuration that specifies which files have to be loaded is located in the file config.php.

Contact form

This project includes a contact form ready for use.
All you need is to configure some constants located in file plugin/contact-form/config.php:

Required data

Constant Description
CONTACT_FORM_TO_NAME name of recipient eg. "My Company"
CONTACT_FORM_TO_EMAIL e-mail address of recipent eg. "mycompany@gmail.com"

Optional SMTP settings

Constant Description
CONTACT_FORM_SMTP_PORT your SMTP port eg. "587"
CONTACT_FORM_SMTP_HOST your SMTP host eg. "smtp.gmail.com"
CONTACT_FORM_SMTP_USER login to your e-mail account eg. "mycompany"
CONTACT_FORM_SMTP_PASSWORD password to your e-mail account
CONTACT_FORM_SMTP_SECURE secure connection protocol: "SSL" or "TLS"

Texts and messages

Constant Description
CONTACT_FORM_SUBJECT subject of e-mail message
CONTACT_FORM_MSG_INVALID_DATA_NAME text displayed when name is empty
CONTACT_FORM_MSG_INVALID_DATA_MAIL text displayed when e-mail address is invalid
CONTACT_FORM_MSG_INVALID_DATA_MESSAGE text displayed when message field is empty
CONTACT_FORM_LABEL_DATA_NAME label for name field
CONTACT_FORM_LABEL_DATA_MAIL label for e-mail field
CONTACT_FORM_LABEL_DATA_MESSAGE label for message field
CONTACT_FORM_SEND_MSG_OK text displayed when e-mail was sent successfully
CONTACT_FORM_SEND_MSG_ERROR text displayed when e-mail cannot be sent

FAQ

Installing template

To install this template upload all files from www folder into HTTP server.
This template is based on AJAX requests (and PHP) and it can't work locally (without HTTP server).

Twitter settings

Open file config.php and set two values: twitter.name (your Twitter nickname) and twitter.count (number of tweets which should be displayed) in $options array: 

$options=array
(
	....
	'twitter'			=>	array
	(
		'name'			=>	'quanticalabs',
		'count'			=>	10
	)
);

Creating new page

Follow in these steps:

1. Open file config.php and add new entry into $page array.

$page=array
(
	....
	'new_page'			=>	array				// hashtag of the page
	(
		'js'			=>	'new_page.js',			// JS file name which will be loaded when the page will be open
		'tab'			=>	'tab-6',			// unique tab ID
		'html'			=>	'new_page.html',		// HTML file name (content of the page)
		'main'			=>	1,				// this options determines if tab should be visible
		'title'			=>	$title.' - New Page'		// page title
	)
);

2. Open file index.php and add new tab:

<!-- Content -->
<div class="content main box-center">

	<!-- Cascade -->
	<div class="cascade preloader">

		<!-- Box menu -->
		<ul class="cascade-menu overflow-fix">

			...
			<!-- New tab -->
			<li id="tab-6" class="blue-info">
				<a href="<?php $Cascade->_getURL('new_page'); ?>">
					<span class="title">New page</span>
					<span class="subtitle">My new page</span>
				</a>
			</li>
			<!-- /New tab -->
			...

		...

	...

...

3. Create all necessary files: /page/new_page.html and /page/script/new_page.js.

Setting meta tags and page title

You can set these data in two ways: for all pages and/or for each page separately:

1. To set meta tags and tittle for all pages open file config.php and set values: title,meta.keywords,meta.description in $options array.

2. To set meta tags and tittle for single page open file config.php and set values: title,meta.keywords,meta.description in $page for selected page.

Changing Google Maps coordinates

To change Google Maps coordinates in contact page open file /page/script/main.js and change values for coordinate variable which you can find in Google Maps section.

Changing tab background

To change the background tab, use one of the following combinations (color + background image):

Available colors: red, blue, lime, green, yellow, orange, purple, turquoise.
Available background images: info, mail, tick, arrow, people, basket, camera, document, briefcase, lightbulb, image-1, image-2, image-3, image-4.

For example: if you want to change the tab with id tab-2 to red, lightbulb please change class name of li - new class name is a combination of names: the color and background image separated by dash: 

<li id="tab-2" class="red-lightbulb">
    <a href="<?php $Cascade->_getURL('portfolio'); ?>">
        <span class="title">Portfolio</span>
        <span class="subtitle">My Works</span>
    </a>
</li>

Changing header background

To change the header of the page, use one of the following classes:

Available classes: background-carbon (default), background-grid, background-mesh, background-wood, background-wood-2, background-wood-3, background-tiles, background-fibers, background-stripes, background-sandpaper. For example:

<body class="background-wood">

Changing social icons

This project includes two type of social icons: small icons (used in footer) and big icons (used at about page). To change it please use one of available classes.
Available classes: social-rss, social-xing, social-skype, social-flickr, social-google, social-twitter, social-facebook, social-linkedin, social-wordpress, social-soundcloud.
